## 🌟 Overview  

This project focuses on **fine-tuning and experimenting with Large Language Models (LLMs)** using advanced techniques like:

- 🔹 LoRA (Low-Rank Adaptation)  
- 🔹 QLoRA (Quantized LoRA)  
- 🔹 OpenAI Router / API-based interaction  
- 🔹 Custom AI app development  

💡 Built to demonstrate real-world **Generative AI + Agentic AI skills**.

## 🛠️ Installation  

```bash
git clone https://github.com/ashjohn9191-droid/DeepSeek.git
cd DeepSeek
```

Create virtual environment:

```bash
python -m venv venv
```

Activate:

**Windows**
```bash
venv\Scripts\activate
```

**Mac/Linux**
```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage  

Run the main app:

